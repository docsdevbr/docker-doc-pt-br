# checkpoint rm

Remove a checkpoint

### Aliases

`docker checkpoint rm`, `docker checkpoint remove`

### Options

| Name               | Type     | Default | Description                               |
|:-------------------|:---------|:--------|:------------------------------------------|
| `--checkpoint-dir` | `string` |         | Use a custom checkpoint storage directory |
